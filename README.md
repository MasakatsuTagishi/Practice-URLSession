# Practice-URLSession
URLSessionの練習

## 概要
URLSessionについて学習し、GitHubAPIから取得したデータを画面に反映しています。

### 環境
- IDE : Xcode 13.4.1
- Swift : Swift 5.6.1
- 開発ターゲット : iOS 15.5

### 動作
1. URLSessionで取得したデータをUILabelで画面表示しています。
